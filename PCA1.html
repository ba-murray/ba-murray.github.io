Principal Components Analysis on Nutritional Information¶

Introduction¶

PCA is fundamentally a dimensionality reduction algorithm, but it can also be useful as a tool for visualization, noise filtering and feature extraction.The data used for the project contains food items and a break down of their nutrients across 23 variables. After cleaning and performing a simple EDA, I will try to reduce the variables of the data set while retaining as much variance as possible.

Conclusion¶

The PCA algorithm preprocessed by scaling and transforming yielded the best result. The Principal Component Analysis using a boxcox tranformation resulted in 23 variables reduced to six components while still accounting for 77% of the overall variance. Components can be interpreted, to find the influence of each variable however it requires deep subject expertise (nutritionist) to enhance the analysis. PCA algorithms are often used prior to a K-Means cluster analysis.
